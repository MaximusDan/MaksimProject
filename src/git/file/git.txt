1. Регимся и захоим в гит под логином github.com
2. Заходим на сайт git-scm.com
2.2 Скачиваем програмку git.exe которая будет контролировать версии. Раздел downloads - windows - качаем. Падает в с-program files - git- bin
2.3 Запускаем Idea. Создаем проект.
2.4 Настраиваем идею.  file-settings-version control-GitHub
2.4.1 Заполняем. Host:github.com
                 login: логин акка на гитхабе
                 password: пароль акка на гитхабе
Все ввели, нажали кнопку Test.

2.4.2 Указываем место положение ранее скаченной програмки git.exe Заходим  file-settings-version control-Git

Прописываем путь (папка может быть скрыта). Нажимаем Test.

2.5 Заливаем проект на GitHub
2.5.1 Заходим в Idea - VCS - Import into Version Control - Share project on GitHub Далее если нужно ввожим комментарии
, жмем Share, далее обязательно вводим коммент, выбираем все папки и жмем Ok.
2.6 Вуаля проект залит в Git.
2.7 В дальнешем если что-нибудь изменили в проекте нажимаем VCS-Commit- Выбираем:Commit and Push
