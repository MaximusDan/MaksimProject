Методы коллекции с их описанием:

get(index) - метод, который достаёт из коллекции элемент, хранящийся на определённом индексе
add(object) - метод, который добавляет элемент в коллекцию.
add(index, object) - метод, который добавляет элемент по индексу в коллекцию.
remove(index) - метод, который удаляет элемент по индексу из коллекции.
remove(object) - метод, который удаляет элемент из коллекции.
contains(object) - метод, который проверяет наличие элемента
addAll(collection) - метод, который добавляет все элементы одной коллекции в другую.

Эти методы вызываются: название коллекции.метод();

public static ArrayList<Working> people = new ArrayList<Working>() {{

}}  - создаем коллекцию как публичную переменную, что бы можно было вызывать, редактировать из любого класса

Удаляем int из коллекции к примеру цифру 2: list.remove(new Integer(2));

Удаляем char из коллекции: list.remove(new Character('a')); или так chars.remove((Character) 'a');

Удаляем double из коллекции: list.remove(3.4);
